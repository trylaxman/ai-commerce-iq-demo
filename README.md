# AI Commerce IQ Demo

A working prototype for an AI-powered commerce analytics and intelligence platform.

## Features

- Executive commerce dashboard
- Revenue and performance analytics
- Customer segmentation
- Product opportunity insights
- Inventory risk analysis
- Campaign ROI tracking
- AI-generated growth recommendations
- CSV / Excel upload simulation
- Store, product, customer, and campaign management

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- LocalStorage demo data
- Mock AI insights engine

## Run Locally

```bash
npm install --legacy-peer-deps --no-audit --no-fund
npm run dev
```

Open:

```text
http://localhost:3000
```

Demo password:

```text
admin123
```

## Purpose

This demo was created to showcase how an internal AI commerce intelligence system can analyze store, customer, product, inventory, and campaign data to generate actionable growth insights.
